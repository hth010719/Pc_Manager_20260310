이미지 파일을 이 폴더에 넣으면 주문창에서 자동으로 표시됩니다.

파일 규칙
- 두 방식 모두 지원합니다
- 방식 1: `상품ID.png`
- 방식 2: `상품ID/image.png`
- 예시: `1.png` 또는 `1/image.png`
- 현재 코드는 `.png`만 읽습니다

상품 ID 목록
- `1` 스팸 마요 덮밥
- `2` 치킨 마요 덮밥
- `3` 김치볶음밥

- `5` 짜계치
- `6` 라볶이
- `7` 신라면

- `9` 참깨라면
- `10` 소떡소떡
- `11` 치킨 가라아게
- `12` 감자튀김
- `13` 만두
- `14` 콜라
- `15` 사이다
- `16` 아메리카노
- `17` 아이스티
- `18` 단무지 추가
- `19` 김치 추가
- `20` 공깃밥 추가
